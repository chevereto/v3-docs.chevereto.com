# Container Bootstrapped

TIP

Use Container Registry for disposable container provisioning. Bootstrapped means that the application layer is provided on container run.

This method for running Chevereto is recommended when you install it in your local computer. Check the repository at chevereto/docker (opens new window) for instructions.
